# AI Prompt Pack for Pipeline Operations

This prompt pack demonstrates how large language models can support operational workflows by turning raw updates into structured insights.

These prompts can be used with any LLM system to streamline reporting, identify risks, and improve decision-making.

---

## 1. Leadership Summary Generator

Use this prompt to convert raw updates into a concise leadership summary.

Prompt:

Turn the updates below into a concise leadership summary.  
Include the following sections:

- Key progress
- Emerging risks
- Decisions required
- Cross-team dependencies

Keep the summary under 200 words.

---

## 2. Risk and Dependency Detection

Use this prompt to surface hidden risks and dependencies in pipeline updates.

Prompt:

Review the updates below and identify potential risks or dependencies that may impact delivery timelines.

For each risk, include:

- Description of the risk
- Severity (Low / Medium / High)
- Recommended next step

---

## 3. Decision Brief Generator

Use this prompt to quickly create a leadership decision memo.

Prompt:

Create a short decision brief using the information below.

Structure the response with the following sections:

- Context
- Options
- Recommended path
- Risks
- Decision needed

Keep the brief concise and focused.

---

## 4. Weekly Pipeline Snapshot

Use this prompt to create a structured pipeline report.

Prompt:

Summarize the initiatives below into a weekly pipeline snapshot grouped by stage.

Include:

- Initiative name
- Owner
- Current status
- Key risks
- Next milestone

---

## 5. Release Notes Generator

Use this prompt to convert internal updates into release notes.

Prompt:

Convert the updates below into customer-friendly release notes.

Include:

- What changed
- Who benefits
- Why it matters

Avoid internal terminology.

---

## 6. Update Simplifier

Use this prompt to clean up messy updates.

Prompt:

Rewrite each update below into a concise two-sentence status report:

Sentence 1: What changed since the last update  
Sentence 2: What is happening next

Highlight any blockers.

---

## 7. Initiative Overlap Detection

Use this prompt to identify duplicated or overlapping work.

Prompt:

Review the initiatives below and identify any overlapping work, duplicate efforts, or potential consolidation opportunities.

Explain why the overlap exists and suggest a resolution.

---

## 8. Meeting Agenda Generator

Use this prompt to prepare for pipeline review meetings.

Prompt:

Create a 30-minute meeting agenda based on the pipeline updates below.

Prioritize:

- Blockers
- Cross-team dependencies
- Decisions needed from leadership
